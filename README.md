# 人工评价指引

### 1. 快速开始

1) 输入 annotation id， 点击'Start'按钮开始评价,一共100条数据，需要为3个不同的模型进行打分，按照不同的指标
2) 给出 6 个评分，点击'Submit'按钮

### 2. 问卷评价任务简介

给定一个主题，围绕该主题生成问卷，评价问卷的质量。

---

### 3. 评价指标
<img width="70%" alt="image" src="https://github.com/LeiyanGithub/questionnaire_evaluation/assets/45895439/f02bcaef-a130-4593-90cf-5f02eddcd967">

人工评价中考虑 4 个方面的文本质量。

- **流畅性（1-5）**: 问卷中句子流畅可读程度比例。

常见的不流畅情况：

1. 句子读不通 ：您认为大学生是否认为自己所以不会做网上课？
2. 句子有歧义 ：您觉得自己的生活费还有盈利？

> 举例说明：

| 评分 | 标准                                        |
| ---- | ------------------------------------------- |
| 5    | 所有句子都流畅. 不流畅问题个数/总问题个数 0-20%                              |
| 3    | 大约一半的句子都流畅. 不流畅问题个数/总问题个数 40-60%                             |
| 1    | 所有句子都难以阅读. 不流畅问题个数/总问题个数 80%-100%                        |

- **多样性（1-5）**: 问卷中句子重复比例。计算方式（独特的问题）/总问题数

举例说明：

| 评分 | 标准                           |
| ---- | ------------------------------ |
| 5    | 几乎没有重复.  重复问题个数/总问题个数  0-20%      |
| 3    | 大约一半以上的句子都出现重复。 重复问题个数/总问题个数  40-60%        |
| 1    | 重复严重。 重复问题个数/总问题个数       80%-100%       |

- **相关性（1-5）**: 问卷中问题与主题不相关的比例
举例说明：

| 评分 | 标准                         |
| ---- | ---------------------------- |
| 5    | 几乎没有不相关问题. 问题与主题不相关个数/总问题个数    0-20%       |
| 3    | 大约一半以上的问题都不相关。问题与主题不相关个数/总问题个数  40-60% |
| 1    | 问题都完全不相关。  问题与主题不相关个数/总问题个数   80%-100%   |

- **合理性（1-5）**: 问卷中选项与问题不匹配、不互斥、不完备
举例说明：

| 评分 | 标准                                 |
| ---- | ------------------------------------ |
| 5    | 几乎没有不合理的选项设置.  选项不合理的问题个数/总问题个数  0-20%   |
| 3    | 大约一半以上问题对应的选项都不合理 40-60%  |
| 1    | 严重不合理。       80%-100%    |

- **整体质量（排序）**: 高质量->低质量: 2 0 1

# High-level

- **客观占比（问卷中客观问题个数/总个数）**: 高质量的问卷应该能是尽可能从客观问题推断主观的看法

- **信息量（问卷中客观问题个数/总个数）**: 高质量问卷应该是能够在填写后收集到更多有用信息，因此，我们计算信息量丰富程度，如果一个问题有多个选项，填写之后能获取更多信息，这种问题被称为高质量问题，总选项个数/问题个数

- **支撑度（仅根据问卷内容生成问卷标题）**: 为每个问卷写一个标题，与原始标题计算bleu值
