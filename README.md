人工评价指引

### 快速开始
1. 输入annotation id， 点击'Start'按钮开始评价
2. 给出6个评分，点击'Submit'按钮

### 问卷评价任务简介
**问卷评价描述**
给定一个主题，围绕该主题生成问卷，让受访者填写。

### 评价指标
人工评价中考虑6个方面的文本质量。

**流畅性（1-5）**: 问卷中句子流畅可读程度比例。

常见的不流畅情况：
1) 句子读不通 ：您认为大学生是否认为自己所以不会做网上课？
2) 句子有歧义 ：您觉得自己的生活费还有盈利？

1: 


| 评分 | 标准                                                          |
|-------|-------------------------------------------------------------------|
| 5     | 所有句子都流畅.                                         |
| 4     | 大多数句子都流畅, 只有少数句子有问题.                  |
| 3     | 大约一半的句子都流畅.                           |
| 2     | 大多数句子都难以阅读, 只有少数句子可以阅读. |
| 1     | 所有句子都难以阅读.                                   |
